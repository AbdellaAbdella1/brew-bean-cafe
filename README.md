# Brew & Bean Café Website

A responsive website for Brew & Bean Café located in Cardiff.

## Features

- Responsive design for all devices
- Accessible navigation and content
- Menu with prices and descriptions
- Location information with Google Maps
- Contact details

## Pages

- **Home**: Welcome section and café features
- **Menu**: Complete coffee and food menu with prices
- **Contact**: Location, map, and contact information
